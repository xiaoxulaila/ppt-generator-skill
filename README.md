# PPT Generator 使用指南

## 简介

PPT Generator 是一个智能 PPT 生成助手，可以根据你的描述自动生成专业、漂亮的 PPT 文件。

## 功能特点

- ✅ **支持所有行业**：商务、教育、科技、医疗、金融、创意等
- ✅ **双语言支持**：中文/英文自动切换
- ✅ **多种风格**：简约、商务、科技、活泼、高端
- ✅ **模板支持**：可以使用你提供的模板
- ✅ **AI 配色**：自动选择最合适的配色方案

## 使用方法

```
用户：帮我做一个关于AI发展趋势的PPT，10页，商务风格

AI：好的！我来帮你生成...

📋 需求确认：
- 主题：AI发展趋势
- 行业：科技
- 风格：商务
- 页数：10页

🎨 配色方案：科技深空（深黑+科技蓝）

📄 大纲设计：
1. 封面：AI发展趋势
2. 目录
3. AI行业概览
4. 技术演进历程
5. 主要应用领域
...

✅ 已生成：~/Desktop/AI发展趋势.pptx
```

## 使用模板

如果你有自己的 PPT 模板，可以这样使用：

1. 把模板文件放到 `templates/` 目录
2. 告诉 AI 使用模板：
   ```
   用模板1帮我做一个产品介绍PPT
   ```

## 常见问题

### Q: 生成的文件保存在哪里？
A: 默认保存在桌面，文件名为 PPT 标题.pptx

### Q: 可以生成多少页？
A: 没有限制，建议 10-20 页效果最佳

### Q: 支持哪些语言？
A: 中文、英文完全支持，会自动识别

### Q: 如何修改生成的 PPT？
A: 用 PowerPoint/Keynote 打开后直接编辑即可
