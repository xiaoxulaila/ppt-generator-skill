## 第一套：牛油果绿ins风竖版PPT

### 核心风格定位
ins风治愈系牛油果绿极简商务/产品展示PPT，适配小红书/公众号长图、竖版产品介绍，主打「低饱和莫兰迪绿+留白+不规则几何」，风格清新、高级、有呼吸感。

### 色彩系统
- 主色：牛油果绿（#A9B388、#8F9779）
- 辅色：米白（#F5F5F0）、浅灰（#E8E8E0）
- 强调色：深棕绿（#5C674F）

### AI生成指令
```
生成ins风牛油果绿竖版产品展示PPT，9:16比例，莫兰迪低饱和牛油果绿为主色，米白为底。
版式：首屏顶部标题栏+产品图占位符；产品模块用深绿通栏+图标栏、弧形底+圆形标题栏；优势模块用方块图+图标栏；详情模块用左图右文+圆形标题；所有模块用不规则波浪线/弧形分割，留白≥30%。
字体：标题思源黑体Bold，正文思源黑体Light。
视觉元素：线性简约圆角图标，圆角矩形占位符，无尖锐直角。
```

## 第六套：多风格潮流创意PPT合集

### 核心风格定位
潮流创意多风格PPT合集，包含「弥散风、孟菲斯风、酸性设计、雕塑风、时尚风、毕业答辩风、作品集风」等多种潮流风格。

### 分风格

1. **弥散风**：低饱和马卡龙色，弥散渐变，网格排版
2. **孟菲斯风**：几何撞色，自由网格，活泼创意
3. **酸性设计**：粉紫渐变，星芒元素，赛博感
4. **雕塑风**：黑白灰，大图+多栏文字
5. **时尚风**：绿色渐变，星芒元素，杂志风
6. **毕业答辩风**：水墨元素，正式规范
7. **作品集风**：黑白灰，网格排版

### AI生成指令
```
生成潮流创意多风格PPT合集，16:9横版，包含弥散风、孟菲斯风、酸性设计等多种风格。
根据选择的风格应用对应的色彩、排版和视觉元素。
```
