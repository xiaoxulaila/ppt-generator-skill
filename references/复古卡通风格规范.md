# 复古卡通风格（Molle手绘风）PPT 设计规范

> 适用场景：艺术创意、手账风格、温馨治愈、复古杂志、个人品牌
> 风格名称：Molle手绘风 / 粗线条涂鸦风 / 复古卡通

---

## 一、核心风格定义

**复古卡通风格**（也叫「Molle手绘风」或「涂鸦风」）是一种温馨、治愈、有手账感的视觉风格：

- 以**奶油黄**为主色调，搭配**米白色**底色
- 所有元素使用**粗黑马克笔轮廓线**勾勒
- 线条**圆润有手绘感**，不做精密几何
- 内部**纯色平涂**，无渐变
- 整体氛围**温馨治愈**，像翻阅手账或复古杂志

---

## 二、配色方案

### 基础色
| 用途 | 颜色 | 色值 |
|------|------|------|
| 背景 | 米白/奶油白 | `#FFFDF5` 或 `#FDF6E3` |
| 主色调 | 奶油黄 | `#F9E79F` |
| 文字 | 深炭黑 | `#2C2C2C` 或 `#1A1A1A` |
| 轮廓线 | 炭黑色 | `#2C2C2C` |
| 辅助色 | 淡粉 | `#F5C6CB` |
| 辅助色 | 淡蓝 | `#AED6F1` |
| 辅助色 | 淡绿 | `#A9DFBF` |
| 辅助色 | 淡橙 | `#FAD7A0` |

### 配色原则
- 奶油黄占 **40-50%**
- 米白背景占 **40-50%**
- 炭黑轮廓线统一所有元素
- 辅助色做点缀，每页不超过 2 种

---

## 三、图形风格规范

### 轮廓线
- 使用 **3-5px 粗黑线**勾勒所有元素
- 线条圆润流畅，有手绘感
- 不追求精准几何，允许轻微不规则
- 转角处圆润处理，无尖锐边角

### 填充
- 内部纯色平涂
- 不使用渐变、阴影、纹理
- 颜色饱和度适中，温馨不刺眼

### 手绘元素
- 黑色马克笔感线条
- 略有粗细变化的线条
- 手写批注风格的文字和箭头

---

## 四、版式设计

### 整体布局：卡片式模块化
```
┌─────────────────────────────────────────┐
│  ┌─────────┬─────────┬─────────┐       │  ← 顶部信息栏（TIME/PLANT/CARTOON）
│  │  TIME   │  PLANT  │ CARTOON│       │
│  └─────────┴─────────┴─────────┘       │
│                                         │
│  ┌─────────────────────────────┐       │
│  │                             │       │
│  │     主内容卡片区域           │       │
│  │     （圆角矩形背景）        │       │
│  │                             │       │
│  └─────────────────────────────┘       │
│                                         │
│  ┌───────────────┐  ┌───────────────┐ │
│  │  副卡片 1    │  │  副卡片 2    │ │  ← 底部可放多个副卡片
│  └───────────────┘  └───────────────┘ │
└─────────────────────────────────────────┘
```

### 页面结构
- **顶部**：横向信息栏（可放日期、主题标签等）
- **主体**：大卡片区域放置主要内容
- **底部**：可选的副卡片或装饰元素

### 圆角处理
- 所有卡片使用 **20-30px 大圆角**
- 按钮、小标签使用 **10-15px 圆角**
- 营造柔和、亲切的视觉感

---

## 五、装饰元素规范

### 猫咪剪影 🐱
- 黑色猫咪剪影是标志性装饰元素
- 可放在角落或卡片边缘
- 姿态多样：坐着、躺着、伸懒腰、舔爪子

### 手写批注
- 箭头、手绘线条
- 感叹号、星号等强调符号
- "NOTE"、"TIP"、"OK" 等手写风格标签

### 复古标签
- 复古机票/票据样式
- 印章风格的小标签
- 贴纸感的效果

### 小装饰
- 星星 ✨
- 爱心 💕
- 小花 🌸
- 圆点图案
- 波浪线
- 复古边框

## 六、字体规范

| 元素 | 字体 | 大小 | 样式 |
|------|------|------|------|
| 主标题 | 手写体/个性黑体 | 40-52pt | 粗体，手绘感 |
| 副标题 | 手写体/个性黑体 | 24-28pt | 正常 |
| 正文 | 圆润黑体 | 16-18pt | 正常 |
| 标签文字 | 手写体 | 12-14pt | 加粗 |
| 注释 | 圆润字体 | 10-12pt | 斜体或正常 |

### 推荐字体
- 中文：**站酷高端黑**、**方正胖娃**、**思源黑体**
- 英文：**Patrick Hand**、**Indie Flower**、**Caveat**

---

## 七、各页面设计要求

### 封面页
- 大标题居中或左对齐，字号 44-52pt
- 奶油黄色块做背景或点缀
- 猫咪剪影装饰
- 手写风格副标题
- 复古标签元素（日期、地点等）

### 内容页
- 卡片式布局，内容分块清晰
- 每个卡片有奶油黄或淡彩色背景
- 内容条目前加手绘风格图标或圆点
- 边角放猫咪或其他装饰

### 过渡页
- 大字标题 + 装饰边框
- 可用手绘箭头引导阅读
- 复古票据/标签做章节分隔

### 结尾页
- "THANKS" 或 "SEE YOU" 大字
- 猫咪告别插画
- 手写风格祝福语
- 整体温馨治愈
