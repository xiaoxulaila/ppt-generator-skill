# 孟菲斯设计风格PPT模板
**整体定位**：16:9横版经典80年代孟菲斯风格PPT模板，主打高饱和撞色、自由几何元素拼贴、粗线条波普艺术融合，风格活力张扬、创意前卫，适配创意汇报、设计分享、潮流主题演讲、品牌提案等场景，全模板统一孟菲斯视觉语言，页面布局丰富且模块化，支持图文、多图、数据、列表、时间轴、流程图等全类型内容展示。

## 一、核心基础规范
1. **尺寸比例**：固定16:9标准横版，所有页面元素适配该比例，无变形、无留白失衡
2. **配色系统**：主调为**高饱和度撞色组合**（明黄、亮粉、湖蓝、草绿、橘红、黑色），以白色为基础底色，几何图形/线条用纯色块填充，无渐变，色彩对比强烈；文字色与背景形成强对比（白色底用黑/深彩色，彩色块底用白/黑色）
3. **字体规范**
    - 标题类：无衬线粗体（如思源黑体Bold/微软雅黑Bold），大字号，字间距1.1倍，部分英文标题全大写（如Memphis Style、CHAPTER 01）
    - 正文类：无衬线常规体（如思源黑体Regular/微软雅黑Regular），字号适中，行间距1.5倍，项目符号用孟菲斯风格小几何图标（圆点/小方块）
    - 辅助文字：无衬线细体（如思源黑体Light），小字号，用于副标题、图表说明、引用来源等
4. **视觉核心**：全页贯穿**孟菲斯经典几何元素**（不规则圆形、三角、锯齿纹、波浪线、粗线条边框、圆点阵列、彩色方块），元素为自由拼贴布局，不对称但视觉平衡，无多余装饰，几何元素仅做边框、背景点缀、分隔符使用

## 二、逐页版式+视觉细节（按页面顺序）
### 封面页
1. 布局：上中下三段式，居中对齐为主
2. 视觉：背景为白色，顶部/底部散落孟菲斯小几何元素（彩色小方块、波浪线、圆点），无满铺背景
3. 内容区：
    - 顶部：超大号主标题「孟菲斯设计风格模板」，无衬线粗体，黑色
    - 中部：副标题「副标题示例 · 创意与活力的视觉呈现」，无衬线细体，彩色（湖蓝/亮粉），加孟菲斯细线条左右点缀

4. 细节：几何元素仅做边角点缀，不遮挡文字，整体留白充足

### 目录页
1. 布局：左侧纵向数字导航+右侧文字标题，左对齐，白色底色
2. 视觉：左侧数字区用**孟菲斯彩色粗线条圆形边框**包裹数字（01/02/03/04），数字为白色，圆形边框为不同高饱和色（明黄/亮粉/湖蓝/草绿）；数字右侧对应章节标题，标题后跟随小孟菲斯几何图标（与左侧圆形同色）
3. 内容区：
    - 顶部：小标题「目录」，无衬线粗体，黑色，加孟菲斯波浪线下划线
    - 主体：4行目录项，每行「彩色几何边框数字 + 章节标题（如章节标题一） + 同色小几何图标」，纵向等距排列
4. 细节：所有元素纵向对齐，行间距统一，几何图标大小一致

### 章节标题页（共4页，版式统一仅文字/配色不同）

3. 内容区：
    - 右半部分：上方英文「CHAPTER 01/02/03/04」，全大写，无衬线粗体，彩色（与左侧几何块主色一致）；下方中文章节标题（如章节标题一），超大号无衬线粗体，黑色
4. 细节：左右分栏比例约1:1，文字区居中对齐，几何块无遮挡文字

## 三、通用视觉细节要求
1. 所有孟菲斯几何元素（线条/图形/边框）**粗度统一**，为2-3pt，无细线条，纯色填充无渐变、无阴影、无透明度
2. 所有页面**留白合理**，无元素满铺，文字区与视觉元素区明确分隔，无遮挡
3. 同类型页面（如章节标题页、数据图表页）**视觉风格统一**，仅配色/文字/图表类型不同，布局和元素大小一致
4. 所有占位符（图片/图表）**预留明确**，带边框/标识，方便后续插入内容，占位符大小与页面比例适配
5. 几何元素的**配色随机但不杂乱**，每套页面固定3-4种高饱和色，不超过5种，保证视觉协调
