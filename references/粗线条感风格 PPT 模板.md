# 粗线条感风格PPT模板 超细节AI生成描述（可直接投喂）
**整体定位**：16:9横版极简粗线条感UI设计公司品牌PPT模板，主打低饱和柔色系、粗线条几何元素、简约留白排版，风格干净高级、简约专业，兼具设计感与商务感，适配UI/UX设计公司、创意机构的品牌介绍、服务展示、客户案例、团队介绍等商务场景，全模板统一粗线条视觉语言，模块化布局清晰，支持图文、列表、流程、案例、数据、团队展示等全类型内容呈现。

## 一、核心基础规范
1. **尺寸比例**：固定16:9标准横版，所有页面元素适配该比例，留白充足无拥挤，视觉呼吸感强
2. **配色系统**：主调为**低饱和柔色系**（米白/浅灰为底色，深灰/炭黑为文字主色，浅蓝/浅绿/浅橙为辅助点缀色），无高饱和撞色，整体色调柔和统一；所有粗线条、图标均为深灰/炭黑，少量辅助色仅用于重点标识，配色极简克制
3. **字体规范**
    - 标题类：无衬线粗体（如思源黑体Bold/微软雅黑Bold），中英混排时英文为无衬线西文粗体，字号层级分明，部分英文标题全大写
    - 正文类：无衬线常规体（如思源黑体Regular/微软雅黑Regular），行间距1.5倍，文字左对齐，简洁易读
    - 辅助文字：无衬线细体（如思源黑体Light），小字号，用于副标题、说明文字、数据标注等
    - 文字颜色：底色为米白/浅灰时，文字统一用深灰/炭黑，无杂色，保证阅读舒适度
4. **视觉核心**：全页贯穿**粗线条设计语言**（线条粗度3-4pt，无渐变、无阴影、无填充，纯轮廓），几何元素为简约粗线条形（圆形、方形、矩形、线条图标），仅做边框、图标、分隔线、装饰点缀使用，无复杂拼贴，视觉极简；所有图标均为粗线条线性图标，风格统一。

## 二、逐页版式+视觉细节（按页面顺序）
### 封面页
1. 布局：上下分段式，居中对齐，米白色底色，大面积留白
2. 视觉：页面边角散落**极简粗线条小装饰**（细粗线条、小圆形轮廓），无满铺元素，仅做轻量点缀
3. 内容区：
    - 上部：英文主标题「Soft Colors UI Design for Agencies」，无衬线西文粗体，深灰色，字号超大
    - 下部：英文副标题「Here is where your presentation begins」，无衬线西文常规体，深灰色（比主标题浅一度），与主标题上下间距适中
4. 细节：无多余元素，粗线条装饰仅在页面边缘，不遮挡文字，整体极简高级。

### 目录页
1. 布局：左侧纵向数字+图标导航，右侧对应中英标题，左对齐，米白色底色，左侧留白，右侧内容区规整
2. 视觉：每个目录项左侧为**粗线条圆形轮廓**（无填充），内部标阿拉伯数字（01/02/03/04），数字旁搭配同风格粗线条小图标（与目录主题匹配），线条均为深灰色
3. 内容区：
    - 顶部：英文小标题「Table of contents」，无衬线西文粗体，深灰色，左对齐
    - 主体：4行目录项，每行「粗线条圆形数字 + 粗线条小图标 + 英文标题 + 中文字标题」（如01 About us / 关于我们），中英文字上下排列，深灰色，字号依次递减
    - 底部：英文短句「Let's start our journey together」，无衬线西文常规体，浅灰色，左对齐，与最后一个目录项间距适中
4. 细节：4个目录项纵向等距排列，圆形图标、数字、文字均左对齐，粗线条元素大小统一。

### 章节标题页（共4页，版式完全统一，仅文字/图标不同）
1. 布局：左侧大数字+英文标题，右侧中文字副标题，左对齐，米白色底色，背景带轻量粗线条纹理装饰
2. 视觉：左侧阿拉伯数字（01/02/03/04）为**超大号粗线条轮廓**（无填充），深灰色，视觉占比大，为页面核心视觉；背景有淡色粗线条几何纹理（浅灰色），轻量不抢镜
3. 内容区：
    - 左侧：超大粗线条数字 + 英文章节标题（如01 About us），英文为无衬线西文粗体，深灰色，与数字右对齐
    - 右侧：中文字副标题（如了解我们的故事与理念），无衬线常规体，深灰色，与左侧英文标题上下居中对齐
4. 细节：数字粗线条轮廓比例协调，中英文字层级分明，背景装饰仅做氛围烘托。

## 四、通用视觉细节要求
1. 所有粗线条元素（轮廓、图标、分隔线）**粗度统一为3-4pt**，无填充、无渐变、无阴影、无透明度，仅纯深灰色轮廓，视觉语言高度统一
2. 所有页面**留白占比≥30%**，无元素满铺，文字区与视觉区、图文区明确分隔，无遮挡，保持极简呼吸感
3. 同类型页面（如愿景/使命/信念页、数据图表页、客户评价页）**版式、视觉风格完全统一**，仅更换文字/数据/图标，保证整体协调性
4. 所有占位符（头像、案例图、视觉图、图表）**均用粗线条轮廓标注**，预留明确插入位置，占位符比例与页面适配，风格与整体粗线条统一
5. 中英混排时，**英文为无衬线西文专用字体**，中文为无衬线中文字体，字号、字重匹配，排版规整，无杂乱感
6. 所有图标均为**原创粗线条线性图标**，与主题高度匹配，无现成彩色图标，保证风格一致性。
