# PPT 模板库

> 将你的 PPT 模板文件 (.pptx) 放到此目录，我来帮你分析并集成到 Skill 中。

## 模板命名规范

```
templates/
├── business-blue.pptx      # 商务蓝色模板
├── tech-dark.pptx          # 科技深色模板
├── fresh-green.pptx        # 清新绿色模板
├── premium-black.pptx      # 高端黑色模板
└── your-template.pptx     # 你的自定义模板
```

## 添加新模板

### 步骤 1：放置模板文件

将 .pptx 文件放到 `templates/` 目录。

### 步骤 2：告诉 AI 分析

```
帮我分析 templates/ 目录下的模板
```

### 步骤 3：AI 自动提取信息

AI 会分析：
- 配色方案（主色、辅色、强调色）
- 布局风格（封面、目录、内容、结尾）
- 适合场景（商务/科技/教育等）

### 步骤 4：更新模板库

AI 会将分析结果添加到 SKILL.md 的模板库中。

## 推荐模板数量

- 至少 3 个模板：商务、科技、创意
- 最多 10 个模板：覆盖不同行业和风格

## 模板要求

- 格式：.pptx
- 尺寸：16:9（推荐）
- 内容：至少包含封面、目录、内容页、结尾页
- 不要加密或密码保护
